## <h1>📍 ApiSyncro By MJlulu02</h1>

- ⚔️ Made for Minecraft Pvp-Faction Server
- 🏆 Synchronisation between Mine server and Faction server

## 📚 Dependence

__Install the dependencies in the terminal :__

    ```
    - npm i body-parser

    - npm i dotenv
    
    - npm i express

    - npm i express-ip

    - npm i mongoose
    ```
